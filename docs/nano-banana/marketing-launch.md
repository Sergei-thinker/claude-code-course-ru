## Обзор

Модуль 3.2.3 завершает ваш продуктовый визуальный инструментарий маркетинговыми и launch ассетами. App store графика, соцсети реклама, launch анонсы - визуализации, которые представляют ваш продукт миру.

**Ключевой вывод:** Маркетинговые визуализации коммуницируют ценность мгновенно. Отличная launch графика может генерировать волнение, которое параграфы текста не могут.

## App Store Feature графика

### Что они такое

Feature графики - героические изображения, которые пользователи видят в app stores. Это первое визуальное впечатление вашего продукта.

### Требования по платформам

| Платформа | Размер | Примечания |
|----------|------|-------|
| **App Store** | 1024×1024 (иконка), различные для скриншотов | Фокус на иконке приложения и последовательности скриншотов |
| **Play Store** | 1024×500 (feature графика) | Широкий banner формат |
| **Обе** | Множественные device скриншоты | Показать ключевые функции |

### Как генерировать

**Базовый промпт:**
```
Создай app store feature графику для [названия приложения].
Показать приложение на [устройстве], выделяя [ключевую функцию/преимущество].
Чистый, профессиональный стиль. [размеры].
```

**Пример - TaskFlow Mobile:**
```
Создай App Store feature графику для TaskFlow Mobile.
Показать приложение, отображаемое на современном iPhone.
Интерфейс показывает управление задачами для операционных команд.
Профессиональный стиль, подходящий для B2B app store листинга.
Тонкий производственный/операционный контекст на фоне.
16:9 соотношение сторон.
```

## Соцсети реклама креативы

### Что они такое

Соцсети реклама креативы - изображения, разработанные для остановки скролла и коммуникации ценности за долю секунды. Они оптимизированы для performance в ленте.

### Спецификации платформ

| Платформа | Рекомендуемые размеры | Примечания |
|----------|-------------------|-------|
| **LinkedIn** | 1200×627 (ландшафт), 1080×1080 (квадрат) | Профессиональная B2B аудитория |
| **Twitter/X** | 1200×675 (ландшафт), 1080×1080 (квадрат) | Быстрое вовлечение |
| **Facebook/Instagram** | 1080×1080 (квадрат), 1080×1350 (портрет) | Визуально-первые платформы |

### Как генерировать

**Базовый промпт:**
```
Создай соцсети рекламу для [продукта].
Сообщение: [ключевая ценностная пропозиция]
Стиль: [визуальный подход]
Включить пространство для headline текста.
[размеры]
```

**Пример - TaskFlow Mobile:**
```
Создай LinkedIn рекламу для TaskFlow Mobile.
Показать before/after трансформацию:
- Левая сторона: Overwhelmed менеджер операций с clipboard, разрозненные бумаги
- Правая сторона: Тот же менеджер уверенно использует TaskFlow Mobile

Сообщение: От хаоса к ясности.
Яркий, привлекающий внимание стиль.
Оставить пространство вверху для headline текста.
Квадратный формат (1:1).
```

### Паттерны рекламных креативов

| Паттерн | Описание | Лучше всего для |
|---------|-------------|----------|
| **Before/After** | Состояние проблемы → Состояние решения | Истории трансформации |
| **Product Hero** | Device/интерфейс центрирован | Анонсы функций |
| **Lifestyle** | Пользователь в контексте | Эмоциональная связь |
| **Data/Stats** | Числа с визуальным влиянием | Построение доверия |
| **Testimonial** | Цитата с человеком | Социальное доказательство |

## Launch анонс графика

### Что они такое

Launch анонс графика - визуализации "мы здесь". Они идут в blog посты, email рассылки, соцсети анонсы, пресс-релизы - везде, где вы объявляете запуск.

### Когда использовать

- День запуска продукта
- Крупные релизы функций
- Обновления версий
- Milestone анонсы

### Как генерировать

**Базовый промпт:**
```
Создай launch анонс графику для [продукта].
Концепт: "Представляем [название продукта]"
Ощущение: [эмоциональный тон]
Профессиональное качество, подходящее для пресс-релиза.
16:9 соотношение сторон.
```

**Пример - TaskFlow Mobile:**
```
Создай launch анонс графику для TaskFlow Mobile.
Концепт: "Представляем TaskFlow Mobile" энергия.
Hero shot приложения - полированный, премиум ощущение.
Launch/анонс энергия - может быть spotlight эффект,
тонкое движение, или чистая reveal эстетика.
Достаточно профессионально для пресс-релиза, достаточно захватывающе для соцсетей.
16:9 соотношение сторон.
```

### Опции настроения

| Настроение | Визуальный подход |
|------|-----------------|
| **Захватывающий** | Motion blur, динамические углы, яркие цвета |
| **Премиум** | Чистый, минимальный, изысканное освещение |
| **Инновационный** | Футуристический, tech-forward, смелый |
| **Надёжный** | Стабильный, профессиональный, reassuring |
