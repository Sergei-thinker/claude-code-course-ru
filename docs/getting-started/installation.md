# 0.1: Установка и требования

- **Время на выполнение:** 15 минут
- **Требования:** Claude Pro или Max подписка

## Обзор

Модуль 0.1 охватывает установку и аутентификацию для Claude Code. Это справочное руководство предоставляет информацию быстрого доступа в дополнение к интерактивному руководству.

**Ключевой вывод:** Claude Code требует Node.js 18+ и платную Claude подписку. Установка занимает менее 5 минут после выполнения требований.

## Требования

| Требование | Детали |
|------------|---------|
| **Claude подписка** | Claude Pro ($20/месяц) или Claude Max. Бесплатный tier НЕ поддерживается. Зарегистрироваться на [claude.ai](https://claude.ai/) |
| **Node.js** | Требуется версия 18 или выше. Скачать с [nodejs.org](https://nodejs.org/) |
| **Операционная система** | macOS или Windows 10+ |

## Как открыть терминал

Все команды в этом руководстве нужно вводить в **терминал** (также называется командная строка). Вот как его открыть:

**Mac:**
1. Нажмите `Cmd + Space` для открытия Spotlight
2. Введите "Terminal"
3. Нажмите Enter

**Windows:**
1. Нажмите клавишу Windows
2. Введите "PowerShell"
3. Нажмите Enter

Вы увидите окно с мигающим курсором - здесь вы будете вводить команды.

## Быстрая установка

### Проверка версии Node.js

**Откройте терминал и введите эту команду, затем нажмите Enter:**

```bash
node --version
```

Если вы видите `v18.0.0` или выше, переходите к установке. В противном случае сначала установите Node.js.

### Установка Node.js (при необходимости)

**Рекомендуется:** Скачайте LTS версию с [nodejs.org](https://nodejs.org/)

**Используя менеджеры пакетов:**

| Платформа | Команда |
|----------|---------|
| **macOS** | `brew install node` |
| **Windows** | `choco install nodejs` |

**Важно:** После установки Node.js полностью закройте окно терминала и откройте новое. Это обновит PATH вашей системы, чтобы она распознавала команду `node`.

### Установка Claude Code

**В терминале скопируйте и вставьте одну из этих команд, затем нажмите Enter:**

**Mac:**
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

**Windows (PowerShell):**
```powershell
irm https://claude.ai/install.ps1 | iex
```

### Проверка установки

**Введите эту команду в терминале и нажмите Enter:**

```bash
claude --version
```

Если вы видите номер версии, установка прошла успешно. Если вы видите `command not found`, полностью закройте окно терминала и откройте новое, затем попробуйте снова.

## Аутентификация

**Введите это в терминале и нажмите Enter:**

```bash
claude
```

При запросе:
1. Выберите "Claude account with subscription" (НЕ Anthropic Console)
2. Завершите аутентификацию в браузере
3. Выберите предпочитаемый стиль текста терминала

После аутентификации вы увидите приветственный экран Claude Code.

## Проверка здоровья

**Внутри Claude Code введите эту команду:**

```
/doctor
```

Это проверяет:
- Целостность установки
- Активную подписку
- Конфигурацию системы
- Общие проблемы

Здоровый вывод показывает зелёные галочки для всех пунктов.

## Устранение неполадок

### Command not found: claude

**Решение:**
1. Полностью закройте окно терминала и откройте новое
2. Если всё ещё не работает, проверьте конфигурацию PATH:

**Mac:** Добавьте в `~/.bashrc` или `~/.zshrc`:
```bash
export PATH=$HOME/.local/bin:$PATH
```

**Windows:** Перезагрузите компьютер (установщик обновляет PATH автоматически)

### Требуется подписка

- Проверьте активную Claude Pro/Max подписку на [claude.ai](https://claude.ai/)
- Выйдите и войдите снова для обновления аутентификации

### Версия Node слишком старая

**Введите это в терминале:**
```bash
node --version
```

Скачайте последнюю LTS с [nodejs.org](https://nodejs.org/), если ниже v18.0.0

### Отказано в доступе

**Mac:**
```bash
sudo curl -fsSL https://claude.ai/install.sh | bash
```

**Windows:**
Запустите PowerShell как Администратор (щёлкните правой кнопкой PowerShell → Запустить от имени администратора)

## Справочник команд
