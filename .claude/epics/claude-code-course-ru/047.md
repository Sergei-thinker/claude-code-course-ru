# Task: Включить GitHub Pages

## Description
Настроить и включить GitHub Pages для публикации сайта курса. Проверить корректность работы docsify и всех ссылок.

## Acceptance Criteria
- [ ] GitHub Pages включен для репозитория
- [ ] Настроена публикация из docs/
- [ ] Сайт доступен по URL
- [ ] Docsify корректно работает
- [ ] Все ссылки работают
- [ ] Поиск функционирует

## Technical Details
### Шаги для выполнения:

1. **Подготовка файлов**
   - Убедиться что все необходимые файлы в docs/ созданы
   - Проверить наличие .nojekyll
   - Проверить index.html и _sidebar.md

2. **Настройка GitHub Pages**
   ```bash
   # В настройках репозитория на GitHub:
   # Settings > Pages > Source: Deploy from a branch
   # Branch: main > /docs
   ```

3. **Проверка после публикации**
   - Открыть URL: https://sergei-thinker.github.io/claude-code-course-ru/
   - Проверить загрузку главной страницы
   - Проверить навигацию по sidebar
   - Проверить работу поиска
   - Проверить несколько внутренних ссылок

4. **Возможные проблемы**
   - 404 на страницах - проверить пути в _sidebar.md
   - Не загружается docsify - проверить index.html
   - Не работает поиск - проверить плагины в index.html
   - Кириллица отображается неправильно - проверить meta charset

### Файлы для проверки:
- `docs/.nojekyll` - должен существовать
- `docs/index.html` - корректная конфигурация docsify
- `docs/_sidebar.md` - правильные ссылки
- `docs/README.md` - главная страница
- `docs/assets/css/custom.css` - подключен в index.html

### Команды для тестирования локально (опционально):
```bash
# Установить docsify-cli
npm i docsify-cli -g

# Запустить локальный сервер
docsify serve docs

# Открыть http://localhost:3000
```
