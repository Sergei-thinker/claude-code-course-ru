# Пример рабочего процесса PM

Представьте себе типичный рабочий процесс PM:

**Утро:**
- Попросите Claude подытожить вчерашние заметки со встречи
- Наблюдайте, как файл с резюме появляется в Obsidian
- Прочитайте, внесите правки при необходимости
- Поделитесь с командой

**День:**
- Попросите Claude составить черновик PRD на основе пользовательского исследования
- Видите, как PRD появляется в реальном времени
- Просматриваете его в Obsidian, общаясь с Claude
- Запрашиваете доработки: "Сделайте раздел 3 более техническим"
- Наблюдаете, как изменения происходят в прямом эфире

**Конец дня:**
- Попросите Claude обработать все задачи на сегодня
- Видите, как появляется организованный список
- Точно знаете, что делать завтра

Это ваша новая реальность! Вы работаете ВМЕСТЕ с Claude, видя всё, всегда контролируя процесс.
