## Тема: Требование мобильного приложения для корпоративного развёртывания

## Описание

Привет, команда TaskFlow,

Мы в процессе развёртывания TaskFlow по всей нашей инженерной организации (150 человек). Однако наша команда безопасности подняла опасения об отсутствии нативного мобильного приложения.

Конкретно:
1. Мобильный веб не интегрируется с нашим MDM (Mobile Device Management)
2. Мы не можем применять политики безопасности на уровне приложения
3. Нет способа удалённо стереть данные TaskFlow, если устройство потеряно
4. Доступ через браузер труднее аудировать

Это блокеры для полного корпоративного развёртывания. Мы можем продолжить с десктопными пользователями, но ~40% нашей команды нуждается в мобильном доступе для дежурных ротаций и реагирования на инциденты.

Каков roadmap для нативного мобильного приложения с корпоративными функциями безопасности?

Роберт
