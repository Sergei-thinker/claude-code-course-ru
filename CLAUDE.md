## Project: Русскоязычный курс Claude Code для PM

Это русская адаптация курса [claude-code-pm-course](https://github.com/Sergei-thinker/claude-code-pm-course).
Оригинальный репо доступен для справки: `~/temp/claude-code-pm-course/`

## Архитектура

- **Сайт**: `docs/` — GitHub Pages + docsify.js (чистый markdown, без сборки)
- **Интерактив**: `course-materials/` — slash-команды, CLAUDE.md скрипты, упражнения
- **CCPM**: `ccpm/` — система управления проектом

## Принципы перевода

1. **Естественный русский** — не машинный перевод. Обращение на «вы».
2. **Английские PM-термины** остаются: PRD, OKR, sprint, backlog, roadmap, stakeholder, MVP, NPS, ARR. При первом упоминании — пояснение в скобках.
3. **Slash-команды** на английском: `/start-1-1`, `/start-2-3` и т.д.
4. **Имена файлов** на английском. Переводится только содержимое.
5. **Код** (Python, bash) не переводится. Переводятся только комментарии.
